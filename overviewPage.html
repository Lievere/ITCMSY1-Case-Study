01 About This Project

Case Background

IT COMSEE Corp. is a mid-sized organization undergoing rapid growth, planning to modernize its IT infrastructure to support its expanding operations. The organization requires a reliable, secure, and scalable computing platform that can handle the demands of a growing workforce.

The management has tasked our team — IT Systems Consultants, Group 7 — to analyze, evaluate, and recommend an appropriate computing platform based on technical, performance, and cost considerations. The total budget for the project is PhP 10,000,000.

This proposal covers six key areas: processor architecture, data storage design, network and I/O technology, virtualization strategy, and performance evaluation — each presented as a dedicated section of this website.

03 Organization Profile

IT COMSEE Corp. — Client Overview

IT COMSEE Corp. is a mid-sized company in active growth. Their current IT setup can no longer keep up with demand — making a full infrastructure overhaul both necessary and timely.

Workforce

80–120 employees across three departments: Administration, Operations, and IT Support.

Authentication

Requires centralized authentication and role-based file sharing across all departments.

Data Security

Needs secure backups and fault tolerance to ensure business continuity during hardware failures.

Expansion

Plans to deploy virtual machines or containers for service isolation and future scalability.

05 Constraints

Budget, Users & Scalability

Every recommendation in this proposal is shaped by three governing constraints. All hardware, software, and design decisions must stay within these boundaries.

Budget Ceiling
PhP 10,000,000
Hard cap across all components — hardware, software licenses, cabling, and implementation. No single category may exceed its allocated share.
User Scale
80–120 Users
All network, storage, and compute capacity is sized for simultaneous peak load across the full 120-user ceiling, not the average.
Scalability
Future-Ready
All infrastructure choices must support organic growth — additional users, VMs, or storage — without requiring a full rebuild.
Static Website
HTML + CSS
This deliverable is a static site only. No backend, no APIs, no frameworks. All content is original and written in the team's own words.

Additional technical constraints: all cited information includes proper references; diagrams are SVG or image-based; JavaScript is used for interactivity only (scroll tracking, counters, accordions).
